# CDN links

In this section, you will find the CDN links (Content Delivery Network) for the alt:V files.
It is advisable to utilize the [Downloads page](https://altv.mp/downloads), where the files are conveniently bundled for you.

The update.json file includes the build number, file locations, and SHA1 hashes, providing valuable information.

:::caution BRANCH placeholder

Ensure to use the appropriate values for ${BRANCH}, which can be release, rc, or dev.

:::
